# Tamagotchi Pet

A small Python/Pygame Tamagotchi-style virtual pet project.

Currently runs on Windows for development. Intended future target is a Raspberry Pi Zero with a Whisplay HAT or similar small display/input setup.

## Requirements

- Python 3.12
- pygame 2.6.1

## Setup on Windows

Create a virtual environment:

```powershell
py -3.12 -m venv .venv
